Welcome to xvortices's documentation!
=====================================

.. image:: https://raw.githubusercontent.com/miniufo/xvortices/master/pics/cylind.jpg

`xvortices` is a python package built on
`xarray <http://xarray.pydata.org/en/stable/>` (starts with an `x`), targeting
at interpolating information of moving vortices from lat/lon gridded datasets
onto cylindrical coordinates.
